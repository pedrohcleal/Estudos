# Capítulo 1::Exercício 2

Os programas Java são compilados em um formato específico, chamado *bytecode*, que representa o conjunto de instruções da máquina virtual Java (**JVM**), que é armazenado nos arquivos de extensão *.class*. Este formato não depende do hardware do computador, nem mesmo de seu sistema operacional.

Com isto, um programa criado em Java, depois de compilado, podem ser executado, sem necessidade de modificação ou mesmo recompilação, em qualquer plataforma computacional para a qual se disponha uma máquina virtual Java, que executa o programa da mesma maneira em diferentes computadores, proporcionando *independência de plataforma*.
